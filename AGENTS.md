# recSOL — Liquid Staking Token (RECTOR SOL)

> The Liquid Staking Token (LST) **$recSOL** — empowers users to stake Solana ($SOL) while retaining liquidity and earning rewards seamlessly. The recSOL token represents staked SOL; over time, recSOL relative to SOL increases as staking rewards accrue to the pool.

## Key Features

- **Yield Accrual:** recSOL/SOL value rises over time as staking rewards add to the pool, benefiting all recSOL holders.
